## Key Learning Takeaways

* **Base Case Mastery**: Learning when and how to stop recursion to avoid `StackOverflowError`.
* **State Management**: Using parameters to pass data (like `sum` or `fact`) through recursive calls.
* **Optimization**: Understanding how to reduce time complexity and stack height using the **Divide and Conquer** approach (as seen in `Recursion_7`).
